Source: GitHub — [`the-starters/starters-webflow`](https://github.com/the-starters/starters-webflow) `v2/` and `slater/`

Code that is version-controlled in the `starters-webflow` repo but does **not** belong to the
3.0 embeds the rest of these docs describe: the **v2 site's** page footer scripts, and mirrors
of externally-hosted (Slater.app) bundles.

These pages exist so nothing in the repo is invisible — they are inventories with deployment
notes, not component walkthroughs.

## What's in this group

- **[V2 Page Footers](v2-footers.md)** (`v2/footers/`) — the live, secure footer logic for the
  v2 Webflow pages (`hirethestarters.com`), CDN-loaded per page via jsDelivr.
- **[Slater Mirrors](slater.md)** (`slater/`) — GitHub-managed copies of the Slater.app-hosted
  contract-form bundles, kept for backup/review; not loaded from the repo.
